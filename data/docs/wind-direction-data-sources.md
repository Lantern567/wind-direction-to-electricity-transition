# Wind Direction Data Sources

本文件记录“气候变化下风向格局演变对风电出力的影响研究”的数据体系。原始大数据不直接提交到 Git；这里只维护来源、用途、字段设计和处理规则。

## 1. 气象风场数据

Core use: 计算主导风向、风向集中度、风向非平稳性、风速分布、有效风能密度和反事实风玫瑰。

Candidate sources:

- 中国气象局国家气象信息中心地面气象站风速风向资料。
- ERA5 再分析风场。
- CMFD 中国区域气象格点资料。
- CMIP6 历史和情景风场资料。

Target coverage:

- Spatial: 全国 2421 个国家级气象站，并可扩展到格点尺度。
- Temporal: 1950-2025 年。
- Frequency: 优先逐时；不足时使用逐日主风向、16 方位频率和再分析资料校验。

## 2. 风电场数据

Core use: 构造风电出力结果变量、工程约束和高暴露组。

Candidate sources:

- 国家能源局风电装机、利用小时、弃风率和发电量数据。
- 中国电力企业联合会分省发电量和利用小时数据。
- 风电项目环评、招标文件、上市公司年报。
- 遥感识别风机位置和风机阵列。
- 风机厂商功率曲线和典型机型参数。

## 3. 地形与下垫面数据

Core use: 控制局地粗糙度、地形通道和海陆风结构对风速风向的影响。

Candidate sources:

- DEM 和坡度坡向数据。
- 土地利用和土地覆盖数据。
- 海陆分布、海岸线和近海水深数据。
- 城市化强度和植被覆盖数据。

## 4. 环流与气候驱动数据

Core use: 解释风向变化背后的气候驱动，辅助区分自然变率和长期气候变化。

Candidate sources:

- 全球平均温度。
- 东亚季风指数。
- AO、NAO、PDO、ENSO 指数。
- 西风急流、海平面气压场和位势高度场。

## 5. 电力系统数据

Core use: 区分气象可发电量损失和实际并网发电损失，并控制弃风和消纳约束。

Candidate sources:

- 国家能源局。
- 国家统计局。
- 中国电力企业联合会。
- 分省电力负荷曲线、跨区输电通道和储能规模资料。

## Processing Notes

- 所有角度变量使用 0-360 度，并在计算差值时使用圆形统计方法。
- 风向扇区优先使用 16 方位；敏感性分析可使用 8 方位和 32 方位。
- 轮毂高度修正、缺测补齐和空间插值必须在脚本中记录方法和参数。
- 结果同时报告气象可发电量损失和实际并网发电损失。
