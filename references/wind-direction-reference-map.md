# Wind Direction Reference Map

本文件用于管理后续正式文献、BibTeX 条目和阅读笔记。正式引用进入 `references/*.bib`，阅读摘要进入 `literature/`。

## Citation Groups

### 1. Climate Wind Direction Change

Purpose: 支撑“风向格局本身正在变化”的背景和数据解释。

Need:

- 长期风向趋势。
- 东亚季风、西风带、冷空气路径和海陆风变化。
- 观测与再分析数据的一致性和不确定性。

### 2. Wind Direction and Wind Farm Output

Purpose: 支撑风向影响 AEP、尾流、排布效率和出力波动的机制。

Need:

- 风玫瑰和功率曲线。
- 风机阵列排布和主轴方向。
- 尾流模型、方向敏感性和偏航控制。

### 3. Nonstationary Wind Resource Assessment

Purpose: 支撑“按历史风玫瑰建设可能过时”的问题设定。

Need:

- 风资源评估中的历史样本窗口选择。
- 气候非平稳对资源预测和项目融资的影响。
- 历史最优、滚动最优和未来鲁棒方案的比较。

### 4. Robust Planning and Adaptation

Purpose: 支撑动态鲁棒风电规划模型和政策含义。

Need:

- 两阶段随机优化。
- 鲁棒风电场布局。
- 储能、电网协同、智能偏航和存量技改。
- 气候适应性基础设施和风险定价。

## Reference Workflow

1. 在 `literature/wind-direction-search-strategy.md` 中记录检索式和筛选标准。
2. 新增正式文献时，先写入 BibTeX 文件，再补充一条阅读笔记。
3. 阅读笔记至少包含研究问题、数据、方法、核心发现、局限和可借鉴位置。
4. 写作时把引用映射到 Result 1、Result 2、Result 3 或讨论部分。
